Variables

Variables are nothing but reserved memory locations to store values. In other word, you are reserving some space in the memory when you are creating a variable

Based on the data type of a variable, the interpreter allocates memory and decides what can be stored in the reserved memory. Thus, storing integers, decimals or characters in these variables can be done by assigning different data types to the variables.

Since Python is a dynamic programming language, thus no explicit variable declaration needed to reserve memory space, while they are automatically done when you assign a value to a variable. The equal sign (=) is used to assign values to variables.

counter = 100  # An integer assignment
miles = 1000.0  # A floating point
name = "Murphy"  # A string assignment

Python Numbers

Number data types store numeric values, which Number objects are created when a value assigned to them and accessed by reference. For example,

var1 = 1

The reference can be removed by using the del statement, ie

del var1[,var2[,var3[...,varN]]]

Python supports four different numerical types -

  • int (signed integers)
  • long (long integers, they can also be represented in octal and hexadecimal)
  • float (floating point real values)
  • complex (complex numbers)
Several examples of numbers in each type

  • A complex number consists of an ordered pair of real floating-point numbers denoted by x + yj, where x and y are the real numbers and j is the imaginary unit.
Python String

Strings in Python are identified as a contiguous set of characters represented in the quotation marks (either single or double quotes). Subsets of strings can be taken using the slice operator ([] and [:]) with indexes starting at 0 in the beginning of the string and working their way from -1 at the end

The plus (+) sign is the string concatenation opeartor and the asterisk (*) is the repetition operator.

str = 'Hello World!'
str[0:5] # 'Hello'
str[6:-1] # 'World'
str +" Earth" # 'Hello World! Earth'
Python Lists

Lists are the most versatile of Python's compound data types, which contains items separated by commas and enclosed within square brackets ([])

To some extent, lists are similar to arrays in C, while the difference is that all the items in a list can be different data type.

Available operators are similar to those on Strings -

list = ['abcd', 786, 2.23, 'John', 70.2]
small_list = [123, 'Sam']
list[0]# 'abcd'
list[1:3]# [786, 2.23]
list[2:]# [2.23, 'John', 70.2]
small_list * 2# [123, 'Sam', 123, 'Sam']
list + small_list # ['abcd', 786, 2.23, 'John', 70.2, 123, 'Sam']
Python Tuples

A tuple is another sequence data type that is similar to the list, which consists of a number of values separated by commas. However, there is also some differences between them -

    Differences between Lists and Tuples :
  • Lists are enclosed in brackets ([ ]), while tuples in parentheses (( ))
  • Tuples can't be updated, said to be a read-only lists

Most of the operations in Python lists can be applied on tuples too, except -

tuple = ('abcd',786,2.23,'john',70.2)
list = ['abcd',786,2.23,'john',70.2]
tuple[2] =1000 # Invalid syntax with tuple
list[2] =1000 # Valid syntax with list
Python Dictionaries

Python's dictionaries are kind of hash table type, which works like associative arrays or hashes that consist of key-value pairs. A dictionary key can be almost any Python type, but usually numbers or strings. Values, on the other hand, can be any arbitrary Python object.

Dictionaries are enclosed by curly braces ({}) and values can be assigned and accessed by using square braces ([ ]). For example -

dict ={} # Declare an empty dict
dict['one'] "This is one"# Assign a key-value pair (key: 'one', value: "This is one")
dict['one'] # Access entry with key 'one'
dict.keys() # Return all the keys in given dict
dict.values() # Return all the values in given dict

Dictionaries have no concept of order among elements, just simply unordered

Flow Controls

Decision Making Statement :

if expression1 :statement1 elif expression2 :statement2 else:statementElse

while-loop :

while expression : true_statement else false_statement

for-loop :

for iterating_var in sequence: true_statement else false_statement

Note : do...while-loop is not supported in Python, while it can be accomplished in logic ways

No. Control Statement & Description
1 break - Terminates the loop statement and transfers execution to the statement immediately following the loop
2 continue - Causes the loop to skip the remainder of its body and immediately retest its condition prior to reiterating
3 pass - Terminates the loop statement and transfers execution to the statement immediately following the loop
